## 🛠️ 贡献指南

### 贡献流程
1. **Fork仓库**
   [![Fork按钮指引图](https://img.icons8.com/fluency/48/000000/code-fork.png)](https://docs.github.com/get-started/quickstart/fork-a-repo)

2. **创建目录**
   ```bash
   CCFD/课程名称/学期_年份/资源类型/
   ```
   - 输入`/`自动生成子目录
   - 在对应位置添加README说明文件

3. **上传文件**
   - 支持拖拽上传（≤25MB/文件）
   - 推荐使用[Git Large File Storage](https://git-lfs.github.com/)管理大文件
